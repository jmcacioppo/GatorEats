# GatorEats
Web app to improve the Gator Dining menu interface.

## Website Link: https://gatoreats.herokuapp.com/

## How to run locally:
- Clone the repository:
```bash
git clone https://github.com/jmcacioppo/GatorEats.git
```
- Run npm install:
```bash
npm install
```
- Start the app:
```bash
node server.js
```
